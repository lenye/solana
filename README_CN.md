# solana pump.fun 代币兑换 API

获取代币的实时价格，购买和出售代币，使用以下 API：

* 价格：https://swap.nanhook.com/pumpfun/price
* 购买：https://swap.nanhook.com/pumpfun/buy
* 出售：https://swap.nanhook.com/pumpfun/sell
